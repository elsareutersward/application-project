Web server

Definition - What does Web server mean?

A web server is a system that delivers content or services to end users over the internet. A web server consists of a physical server, server operating system (OS) and software used to facilitate HTTP communication.

Web server explained:

A web server is a program that uses HTTP (Hypertext Transfer Protocol) to respond to a users request by delivering content and services. Whenever a user wants to access a file via their browser, the browser sends a request to the web server hosting the file via HTTP. When the request reaches the correct web server, the server accepts and processes the request by sending back the requested document to the browser through HTTP.

Three common web servers:

  1. Apache (the most widely-installed Web server)
  2. Microsoft's Internet Information Server (IIS)
  3. Nginx
